Practice Exam 1

1. Which of the following best describes the relationship between testing and quality assurance (QA)?

Correct Answer: B. Testing is a product-oriented, corrective approach, while QA is a process-oriented, preventive approach.

2. Which of the following is an example of horizontal scaling in the AWS Cloud?

Correct Answer: D. Adding more EC2 instances of the same size to handle an increase in traffic.
